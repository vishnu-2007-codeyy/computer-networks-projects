# Multi Client Chat Application

A Python socket based chat application with Tkinter GUI.

## Features

- Multiple clients connect to one server
- Real-time messaging
- Graphical interface using Tkinter
- Message timestamps
- Threaded server
- Graceful disconnections

## How to Run

Start Server

python server/chat_server.py

Start Client



python client/chat_gui.py

## Technologies Used

Python
Sockets
Threading
Tkinter
